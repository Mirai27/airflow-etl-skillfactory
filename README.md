Репозиторий содержит финальный проект и практические работы по автоматизации ETL-процессов с использованием Apache Airflow (3.2.1), PostgreSQL и S3 (MinIO).

## Финальный проект (Data-Driven DAG)

Основное решение финального задания находится в папке `dags/` в файле `skillfactory_dynamic_dag.py`.

Реализована динамическая генерация задач (Dynamic Task Generation) на основе единого конфигурационного Python-словаря (`CONFIG`). Этот подход позволяет аналитикам и разработчикам добавлять новые агрегаты и таблицы в базу данных без изменения самого кода DAG.

### Ключевые особенности решения:
* **Автоматический DDL:** Пайплайн сам проверяет наличие необходимых таблиц в СУБД и создаёт их при первом запуске (`CREATE TABLE IF NOT EXISTS`).
* **Абсолютная идемпотентность:** Реализован отказоустойчивый паттерн `Delete-Insert` с привязкой к логической дате Airflow `{{ ds }}`. Исторические данные защищены от дублирования при повторных ручных или автоматических перезапусках.
* **Опциональный экспорт:** На основе флага `need_to_export` в конфигурации пайплайн динамически решает, нужно ли добавлять в граф задачу выгрузки отчета в S3-хранилище в формате CSV.
